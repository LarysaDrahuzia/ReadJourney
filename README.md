## RentalCar Application

## Description

**RentalCar** is a web application for car rentals that allows users to browse a
catalog of cars, apply filters, save favorites, and book vehicles.

---

## Key Features

- View a list of cars with detailed cards
- Filtering by brand, price, and mileage
- Pagination with a **Load More** button
- Adding cars to the favorites list with persistence in **localStorage**
- Booking form with validation and notifications

---

## Installation

Clone the repository:

```bash
git clone https://github.com/LarysaDrahuzia/RentalCar

Install dependencies:

npm install
```

## Configuration

Make sure that the environment variable REACT_APP_API_BASE_URL points to:

https://car-rental-api.goit.global

## Running the Project

The project is built with Vite and React. Axios is used for API requests.

Start the development server and open in your browser:

http://localhost:5173

## Usage

On the /catalog page, browse the list of cars.

Use the filters at the top to refine your search.

Click "Load More" to fetch additional cars.

Click ❤️ on a card to add a car to favorites.

Clicking "Read more" opens the car details page with a booking form.
